# Implementation-of-RL-algorithms
Agent is trained in gym-Minigrid env.


gym-Minigrid https://github.com/maximecb/gym-minigrid


The Agent is trained using 3 algorithms:
1. Sarsa 
2. Sarsa Lambda
3. Q-Learning

The agent was trained on 4 environments:

1.6x6 Empty room
2.8x8 Empty room
